## CSS Classes for Responsive Behavior

```css
/* Hide/Show by device */
.hide-mobile    /* Hidden on mobile */
.show-mobile    /* Visible only on mobile */
.hide-tablet    /* Hidden on tablet */
.show-tablet    /* Visible only on tablet */

/* Responsive utilities */
.container-responsive  /* Auto-adjusting container */
.grid-responsive       /* Auto-adjusting grid */
.flex-responsive       /* Stacks on mobile */
.stack-mobile          /* Vertical stack on mobile */
.full-width-mobile     /* Full width on mobile */
.touch-target          /* 44x44px minimum */
```

## Performance Tips

1. **Reduced Motion**: Automatically enabled for users who prefer it
2. **Touch Devices**: Optimized for one-handed use
3. **Small Screens**: Simplified layouts reduce cognitive load
4. **Bottom Sheet**: Easier to reach on large phones
5. **Responsive Images**: Scale appropriately for device
